Langton's Ant Simulator

What is it?

Langton's Ant Simulator (LAS) is a program that simulates the Turing machine, Langton's ant. It was made by Daniel Shaw. It is written in Java.

Usage

You can either run this program as you would normally for the default configuration, or use command line arguments to modify the settings.
When using arguments, surround the argument in quotes. For example:

$ java -jar Ant.jar "--speed 10"

All of the arguments and what they do:
Argument Parameters What it does
-h None Print help
-v None Display version
--tile-size <Integer> Set the tile size (must be greater than 8 and less than 32)
--speed <Integer> Set the speed of the simulation (one iteration every n milliseconds). Has to be greater than 0
--random None Generate a random grid
--width <Integer> Set grid width to n, must be greater than 100
--height <Integer> Set grid height to n, must be greater than 100
--ants <Integer> Set number of ants
Note: You can press escape at anytime during the simulation to force stop the simulation.
Once the simulation has stopped, you can dump the data from the simulation to a file on your desktop.
